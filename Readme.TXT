程序使用步骤:
1、把你的目标原图放置在某个目录下；
2、运行MATLAB程序（如果报错可能是一些GUI的支持包还没安装，请自行google/百度相关报错提示，再安装好相应支持包）；
3、将“原图目录”和“原图名”复制于GUI相关框内（示例“原图目录”为'.'，即当前目录，示例“原图名”为'heart.png'，是我给的示例图片）；
4、接下来可点击“载入图片”按钮试着载入图片，点击“载入并运行”则会运行G-S算法进行DOE迭代设计，可见四张子图，分别为光源强度分布（默认为匀强光源）、迭代X次后的DOE相位分布、迭代X次后的成像效果、目标成像图样；
5、勾选“负片”可得负片效果、勾选“存图”则会把设计结果存于“原图目录”中（示例所得"_heart.pngIterationTimes101---500-500DOEPhase.png"即为输入仪器中的DOE相位分布图样）；“波长”、“焦距”、“DOE像素边长”、“迭代次数”等参数可选择性改变。
